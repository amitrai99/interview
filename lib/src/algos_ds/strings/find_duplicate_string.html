Find duplicate characters in a String

All such problems fall under the Element distinctness problem domain See https://en.wikipedia.org/wiki/Element_distinctness_problem

It is not possible to find distinct elements in O(n). The minimum is O(nlog(n))

1. Can be done easily using hashmaps.

2. Another approach could be using a array with each position representing ascii char. Good for ascii or ISO_latin only. UTF is too large for this.
