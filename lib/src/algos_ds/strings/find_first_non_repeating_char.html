Find the first non-repeated character in a String

Approaches

There can be multiple approaches.

  1. Create a hash for each char in the string where key is char and value is a boolean or number.
    Iterate over the original string and use the char in it as key in hash, to find the first non-repeating char.
    This method is very performant. It makes only 1 O(n) pass over the string followed by O(n/2) pass over hashmap in worst case.
    Worst case of this method is when all the chars are duplicates except the last one.
