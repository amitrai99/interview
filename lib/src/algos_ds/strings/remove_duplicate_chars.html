Remove duplicate characters from String

This can mean 2 things. We either remove all the duplicates or just keep one instance of each char

Approaches

  1. Use a map to count occurence. Complexity: O(n)
    In first loop create map
    In second loop check each char as key in map. If count > 1, delete char.
  2. Use an array 128 length for ascii chars. Complexity: O(n)
    Iterate and Read ascii code of char at location i. Update the count of char in array at index i.
    Iterate second time and delete all chars with count > 1
