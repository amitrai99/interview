Reverse a String using iteration and recursion.

This is only for pure reversal NOT words reversal.

Note: Cannot be done in-place in JavaScript.

  1. Iterative:
    Create char array.
    Loop over array swapping the element from start index with end index.
    Break look when start index >= end index.
  2. Recursive:

    Base case: when start index >= end index.
    Create char array.
    Recursive method takes three parameters, arr, start index and end index.
    On first call pass array, 0, arr.length - 1
    Call the same method with arr, ++start, --end
