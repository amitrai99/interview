Given two strings, find if first string is a subsequence of second

A subsequence is a sequence that can be derived from another sequence by deleting some elements without changing the order of the remaining elements (source: wiki). Subsequence is NOT substring.

Analysis

  1. Iterate over subsequence.
  2. If find a char, move to next char in subsequence. Repeat till you have found all chars in subsequence or the end of String.
