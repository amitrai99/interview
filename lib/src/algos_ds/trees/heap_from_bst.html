Create heap from a binary tree

This question is for testing your understanding of binray tree and another tree data structure called heap. Important point to remember is that we need to convert a Binary Tree which is not necessarily a Binary Search Tree.

What is a heap?

Heap is a type of tree based data sturcture with specific propery. There are two types of heap:
(1) Max heap the parent node is always greater than child nodes.
(2) Min heap the parent node is always smaller than child nodes.

To create a heap from a binary tree

  1. Traverse the binary tree and store the nodes in an array.
  2. Sort the array.
  3. Create a binary tree from the array. Every node has it's left at 2i + 1 and right at 2i + 2.
