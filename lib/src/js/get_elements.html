How many ways to get to an element in JavaScript

There are 6 ways:

  1. document.getElementById('some-id')
  2. document.getElementsByClassName('.some-class')
  3. document.getElementsByTagName('p')
  4. document. getElementsByTagNameNS("http://www.w3.org/1999/xhtml", "p")
  5. document.getElementsByName('firstname')
  6. document.querySelector('.my-class')
  7. document.querySelectorAll('.my-class')
